
# IMDB
 This is a React app which displays list of top   20  imdb movies 


## Prerequisites

Node and npm installed
MongoDB installed
## Installation

```bash
  git clone https://github.com/ABW1729/imdb.git
  cd imdb
  npm install 
  cd backend
  npm install
```
    
## Backend

Test Data is present in backend/src/testData.js  
You can modify data as per requirements  
To add data in your local mongodb database run  
```bash
cd backend/src/db
node index.js
```

To start Backend server
```bash
cd backend
npm run dev
```
This will start development server on localhost:5000



## Frontend

To start Frontend run from root of project
```bash
npm run dev
```
This will start frontend on localhost:3000
